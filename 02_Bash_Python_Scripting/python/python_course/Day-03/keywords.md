Python keywords Day-03 
  Understanding how to properly use this pyhton keywords and when to use them 

Conditional-logic: 
  - if, else, elif
  Used to make decisions in code.
  

Loop:
  - for 
   iterates throught list of the files, servers
   - while
   keep retrying until a condition is met 

Error-Handling
   - try 
     use to catch error in code and prevent it from crashing
   - except 
     catch what kind of error

Funcions
    - def 
      defining function
    - return 
      returning a value from a function 

Importing-modules
    - import 
      use to import functions from python built-in modules or custom modules using from 
    - from 
     Imports specific functions from a module
    - as 
      Creates an alias/nickname for imported modules

Common-Values
    - True
      Boolean values for conditions
       Example: `is_production = True`
  
    None
     Represents "no value" or empty
     Example: `config = None` (will be assigned later)

DevOps Notes
- Use try/except when running system commands
- Use for loops for multiple servers/files
- Create reusable functions with def
